Concepto central

¿Qué es la organización digital?

Ordenar

Ubicar cada archivo en una carpeta lógica para poder encontrarlo cuando se necesite.

Nombrar

Asignar nombres claros a los archivos para saber qué contienen sin tener que abrirlos.

Conservar

Guardar evidencias, recursos y productos de forma que puedan reutilizarse y revisarse después.

Organización digital no es solo “guardar cosas”. Es construir una forma ordenada de estudiar, trabajar y entregar evidencias.
Casuística 1

Mini caso: el archivo que nadie encuentra

Situación

Un estudiante realiza su actividad, pero la guarda con el nombre "documento final". Después descarga otra guía, la nombra "nuevo archivo" y al final del día no sabe cuál debe subir.

¿Qué está fallando?

  • El nombre no identifica la clase ni la actividad.
  • Los archivos quedan mezclados.
  • La búsqueda posterior se vuelve lenta y confusa.

¿Cómo se corrige?

  • Usar una convención fija de nombres.
  • Guardar en la carpeta de la clase correspondiente.
  • Evitar nombres genéricos o ambiguos.
Ejemplo correcto: cl2_juanperez_tablabasica.xlsx
Casuística 2

Cómo se organiza una carpeta de trabajo

Estructura recomendada del estudiante

  • TICs_NombreApellido
  • Clase_01
  • Clase_02
  • Clase_03
  • Recursos_personales

Beneficios de esta estructura

  • Facilita la búsqueda
  • Evita mezclar actividades
  • Permite ver el avance por sesión
  • Ayuda a entregar con más seguridad
Error frecuente: guardar todo en la carpeta principal o dejar archivos sueltos en Descargas. Eso hace que el trabajo se pierda o se duplique.
Drive en la práctica

Acciones básicas que debo dominar en Google Drive

Crear

Crear carpetas y subcarpetas según la clase o el tipo de recurso.

Renombrar

Modificar el nombre de un archivo o carpeta para que quede claro y útil.

Mover

Pasar un archivo a la carpeta correcta cuando quedó mal ubicado.

Localizar

Encontrar rápidamente un archivo usando la estructura creada o el buscador.

Antes de usar herramientas más complejas, el estudiante debe dominar estas acciones básicas. Ese es uno de los objetivos reales de la Clase 2.
Entrada a Excel

¿Qué es una hoja de cálculo y para qué sirve?

Definición simple

Una hoja de cálculo es una herramienta que permite organizar información en filas y columnas para visualizarla, editarla y trabajarla con orden.

Uso en este módulo

Nos servirá para construir tablas, registrar datos, presentar información clara y, más adelante, aplicar funciones y procedimientos propios del área.

Organizar Escribir Ordenar Visualizar
Conceptos básicos

Fila, columna, celda y rango

Fila

Conjunto horizontal de celdas. Se identifica con números.

Columna

Conjunto vertical de celdas. Se identifica con letras.

Celda

Espacio individual donde escribimos datos. Ejemplo: A1.

Rango

Conjunto de varias celdas seleccionadas. Ejemplo: A1:D10.

Estos cuatro conceptos son la base. Si el estudiante no los reconoce con seguridad, luego se le dificulta avanzar a fórmulas, funciones o análisis de datos.
Casuística 3

Mini caso: una tabla mal hecha vs una tabla bien construida

Tabla mal construida

  • No tiene encabezados.
  • Los datos están mezclados.
  • No se distingue título de contenido.
  • Columnas muy estrechas o desordenadas.

Tabla bien construida

  • Tiene encabezados claros.
  • Cada dato ocupa su lugar.
  • Se puede leer rápidamente.
  • El formato apoya la comprensión.
La hoja de cálculo no se usa solo para “llenar cuadritos”. Se usa para dar orden, lectura y estructura a la información.
